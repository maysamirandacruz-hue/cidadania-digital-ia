# Cidadania Digital: Identificando Deepfakes

## Objetivo do Site
Este site foi desenvolvido para conscientizar a comunidade escolar sobre os perigos das deepfakes e da desinformação automatizada por IA, trazendo dicas e um quiz interativo.

## Uso de Inteligência Artificial
- **Textos de apoio:** Gerados pelo ChatGPT.
- **Prompt utilizado:** "Crie um texto explicativo curto e focado em estudantes sobre o que são deepfakes e seus riscos para a sociedade."
